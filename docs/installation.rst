Installation

Ensure the following packages are installed:

NumPy
SciPy
Matplotlib
Scikit-learn
Networkx
FLANN (pyflann)
SLEPc4py
PETSc4py
MPI4Py (for parallel processing)
Install Scaman using conda:

conda create --name scaman_env --file requirements.txt --channel conda-forge
